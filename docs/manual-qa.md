# 수동 QA 체크리스트 (Manual QA)

본 문서는 DB/DC 퇴직연금 전환 시뮬레이터의 출시 전 수동 검증 시나리오를 정의한다.
자동화(4-gate)가 커버하지 못한 영역을 수동으로 검증하여 `v0.1.0-rc.1` → `v0.1.0` 전환 조건을 충족한다.

> 본 PR 13은 체크리스트와 결과 기록 형식만 제공한다.
> 실제 수동 QA 실행 결과는 별도 파일 `docs/qa-results-YYYY-MM-DD.md`에 별도 세션에서 기록한다.

---

## 1. 목적과 범위

- 자동화가 커버하지 못한 출시 전 수동 검증
- 18개 시나리오에 대한 단계·기대 결과·체크 항목 제공
- 브라우저 매트릭스 검증 (Chrome, Edge, Safari)
- 모바일 뷰포트 검증 (375px)
- 인쇄/PDF 실제 출력 검증

**범위 외**:
- 자동화 가능한 회귀 테스트 추가 (별도 PR)
- 성능 부하 테스트
- 크로스 브라우저 전면 매트릭스 (최신 1개 버전만)
- 접근성 심화 검사 (WCAG 전체 준수 감사)

### 4.7 YEARLY_CUSTOM (연도별 직접 입력)

> **PR 15 처리**: v0.1.0 N/A/Deferred에서 **active 전환**. PR 15B가 UI·검증·단위테스트를, PR 15C가 인쇄 보고서·공유 안내·E2E·문서를 담당.
> - 계산 엔진(`src/calculator/salary-path.ts`)·단위 테스트는 변경 없이 재사용. UI만 `AdvancedSalarySection` 안에 노출.
> - URL 공유는 기본적으로 `yearlySalaries`와 고급 임금 설정을 제외한다. 사용자가 체크박스로 명시적으로 선택한 경우에만 포함한다.

단계:
1. `고급 임금 시나리오` details 펼침 → 임금 경로 모드 드롭다운에서 **연도별 직접 입력** 선택
2. 남은 근속연수(`n`)만큼의 연도별 연봉 입력 행이 표시되는지 확인 (1년차 … n년차, `n` 클 때 스크롤 박스)
3. **현재 연봉으로 채우기** 버튼 클릭 → baseline(현재 연봉 × (1+임금상승률)^t) 값으로 n행 채워지는지 확인
4. 결과 카드가 갱신되고 **고급 임금 시나리오 적용 중** 뱃지가 표시되는지 확인
5. 행 하나를 비우면 → `연도별 연봉을 확인해주세요:` 에러 메시지 + 결과 패널 계산 보류 안내
6. `남은 근속연수`를 늘리면 기존 입력은 보존되고 새 칸만 추가되는지 확인 (줄이면 꼬리 잘림)
7. 체크하지 않고 공유 링크 복사 → URL에 `advanced=`와 `salaries=`가 없는지 확인
8. `고급 임금 설정도 공유 링크에 포함` 체크 후 복사 → 새 탭에서 모드·연도별 연봉·평균임금이 복원되는지 확인
9. 인쇄(PDF) → 입력 요약에 `연도별 직접 입력 — n년치 (첫 … / 마지막 …)` 요약줄 표시 확인

기대 결과:
- 모드 선택/해제 시 n개 행이 정확히 표시/숨김
- 검증 통과 시에만 결과 계산 (빈/0/음수/과다/길이 불일치 → 결과 보류)
- 기본 공유 링크에는 연도별 연봉 미노출, 명시적 포함 링크에는 동일 값 복원, 인쇄 보고서에는 요약줄 표시

## 6. 결과 기록 형식

실제 수동 QA 결과는 별도 파일에 기록한다.

- 파일명: `docs/qa-results-YYYY-MM-DD.md`
- 작성 시점: PR 13 완료 후 별도 QA 세션
- 각 시나리오별:
  - 결과: PASS / FAIL / N/A
  - 비고: 관찰 내용, 스크린샷 경로(선택)
  - FAIL 시: 재현 단계, 환경, 우선순위

**결과 파일 예시 구조**:

```markdown
# 수동 QA 결과 (YYYY-MM-DD)

- 환경: Chrome XXX, Edge XXX, Safari XXX
- 기반: v0.1.0-rc.1

## 4.1 기본 샘플 입력
- 결과: PASS
- 비고: 기본값 로드 정상

## 4.15 인쇄/PDF 보고서
- 결과: FAIL
- 재현: 인쇄 버튼 클릭 → 대화상자 미열림 (Safari)
- 우선순위: 치명
```

## 7. FAIL 처리 절차

- **자동화 가능**: 본 PR 13 범위 외, 신규 PR에서 회귀 테스트 추가
- **자동화 불가**: 본 문서에 회귀 테스트 케이스로 추가
- **치명 (계산 오류·금지 문구 노출·서버 전송)**: 즉시 `v0.1.0-rc.1` 철회, 핫픽스 PR
